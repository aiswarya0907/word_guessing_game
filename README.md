# word_guessing_game
An interactive Python-based guessing game featuring multiple difficulty levels and categories to create a dynamic and engaging gameplay experience. The project demonstrates core programming concepts including randomness, data structures, and user interaction design.

Features:
1. Three difficulty levels: Easy, Medium, and Hard
2. Multiple game categories with different guessing challenges
3. Randomized questions using Python’s random library
4. Dictionary-based structure to organize categories, levels, and answers
5. Input validation with real-time feedback for players
6. Replayable gameplay with increasing challenge levels

Technologies used:
1. Python
2. random library
3. dictonaries
